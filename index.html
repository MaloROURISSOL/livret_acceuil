Bienvenue, ce site a été réalisé afin de vous communiquer toutes les informations nécessaires pour que votre séjour se déroule dans les meilleures conditions.

Les infos pratiques :


Le code wifi est sur le tableau électrique, à droite de la télévision.

L’aspirateur, la serpillère, l’étendoir à linge sont rangés dans l’espace en dessous de l’escalier (il y a même une lumière pour y voir plus clair !).

Les produits ménagers sont sous l’évier.

Le canapé se déplie si vous avez besoin d’un couchage supplémentaire. Vous disposez d’une serviette et d’un gant de rechange dans l’armoire.

Je vous serai reconnaissante de laisser l’appartement relativement propre : passer l’aspirateur, mettre le linge de maison utilisé par terre dans la salle de bains et vider vos poubelles.

A votre départ, pensez à récupérer votre nourriture dans le frigidaire. Un check-out sera effectué lors de la remise des clés. Merci de me signaler toute casse ou dysfonctionnement.

Les points d’attention :


Le garde-corps de la mezzanine est fait maison, merci de ne pas lui mener la vie dure ! Pas de produits anticalcaires ni de vinaigre pour laver le sol, les murs et la douche : le travertin serait très contrarié. Vous pouvez utiliser du liquide vaisselle ou du produit pour sol (ex : le Carolin sous l’évier).
